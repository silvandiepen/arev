# Data & Assets

The foundational reference layer for `arevdata`.

This section groups the package's country, city, language, phone-code, and currency datasets together with the flag asset helpers they depend on.

## Included pages

- Country records and ISO lookups
- Dialling-code selectors and shared prefixes
- Capital and major-city data
- Subnational states and provinces
- Continent and currency metadata
- Language catalogs and locale mappings
- Hosted flag asset metadata
