Typed astronomy reference data for lightweight interfaces.

This section covers date-aware sun and moon helpers, curated Solar System and star metadata, and timezone mappings that fit the rest of the package: lightweight, typed, browser-friendly reference data with practical lookup helpers.

## Included pages

- Sun and moon helpers for phase and season lookups
- Planet and major-moon reference data
- Notable stars and Milky Way reference data
- IANA timezone mappings with map-ready coverage data
